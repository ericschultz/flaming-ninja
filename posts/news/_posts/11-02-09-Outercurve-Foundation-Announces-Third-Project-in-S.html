Wakefield, MA. February 9, 2011 - The Outercurve Foundation today announced the acceptance of the Network Monitors Experts (NMExperts) project into the Systems Infrastructure and Integration Gallery.  NMExperts, contributed by Microsoft, the Foundation’s founding sponsor, joins the NetMon Parsers project, the anchor project in the SIIG Gallery.

The open source NMExperts project includes three ‘experts’, tools that enable network administrators to analyze raw network data based on capture files and ETL logs.  The tools can be launched from the Network Monitor User Interface and also from other network user interfaces. The NMExperts project is being developed under the New BSD license.

The NMExperts project supports interoperability by providing open source tools that analyze raw network data for a number of network monitoring APIs, including, but not limited to, the Microsoft Network Monitor API.  

“We are pleased to accept a third project focused on increasing interoperability between Windows and a wide range of technologies and platforms,” said Paula Hunter, Executive Director, Outercurve Foundation. “We encourage corporations and open source developers to contribute projects to the Foundation that support interoperability, increase understanding and support the goals of the open source community.”

The Outercurve Foundation has three galleries and 11 projects. Galleries include the ASP.NET Open Source Gallery (five projects), the Systems Infrastructure and Integration Gallery (three projects) and the recently-announced Research Accelerators Gallery (three projects.)

For more information on NMExperts, the SIIG gallery or the Outercurve Foundation please visit www.Outercurve.org.

About The Outercurve Foundation

The Outercurve Foundation is a not-for-profit foundation providing software IP management and project development governance to enable and encourage organizations to develop software collaboratively in open source communities for faster results.  The Outercurve Foundation is the only open source foundation that is platform, technology, and license agnostic.
