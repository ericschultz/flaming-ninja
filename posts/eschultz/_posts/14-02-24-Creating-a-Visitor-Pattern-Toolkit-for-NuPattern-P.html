Make sure to read part 1 and watch its tutorial

My last blog entry and tutorial showed the basics of using my Visitor Pattern toolkit for Outercurve NuPattern. Let's move on to the basics of how a NuPattern toolkit is constructed using my example visitor pattern toolkit.

First we need to make sure to NuPattern installed in Visual Studio 2013. You can do this a few different ways:

  • Inside Visual Studio go to the Tools menu, select "Extensions and Updates" and search for NuPattern
  • Download and run the NuPattern VSIX file from Visual Studio Gallery online
  • Download and run the NuPattern VSIX file from NuPattern's website at http://www.nupattern.org

After completing the installation of NuPattern, you can clone or fork the Visitor Pattern solution code on GitHub or download a ZIP of the source. Open your copy of the solution and you'll be ready to follow along to the tutorials.

The first tutorial gives a quick overview of the NuPattern user interface, views and the pattern definition editor.

The second tutorial covers creating new elements in the pattern definition editor, variable properties, launch points, automation and how to build, debug and distribute your toolkit. That might sound like a lot, but the tutorial is only about five minutes long.

Of course, Outercurve NuPattern can be used for a lot more than just building a simple visitor pattern toolkit. NuPattern toolkits can have a documentation pane in Visual Studio that updates along with the user's current action, perform automation when a particular type of item is dropped on the Solution Builder, multiple views, extension points so downstream users can extend your toolkit, a MUCH cleaner Visual Studio API and more. All those features may sound daunting but don't worry: if you ever get stuck, the NuPattern team is very friendly and always willing to help on their forums.

NuPattern won't be the ideal solution to every development challenge, but if you want to automate tedious, error-prone tasks related to a pattern in Visual Studio, I think NuPattern is a great tool to use.
